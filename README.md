## How to add my information
Create a **New issue** in [Issues](https://github.com/oim3640/Projectinder/issues), with **your full name as issue title**, including but not limited to the following information:

- Name:
- Preferred Contact Method:
- Available Time for Project: 
- Project Teaming Preference: 
- Project ideas:
- Anything else about you:

Check out the [issue example](https://github.com/oim3640/Projectinder/issues/1).
